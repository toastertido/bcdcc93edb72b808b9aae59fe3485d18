Apple TV [Lost Media]

If you have an Apple TV with this game installed, please contact us with the contact info linked in the Credits page.

This version of the game is lost media, meaning that the files for this version of the game are not publicly available. Ripping apps from this device in .ipa or .tipa format is very difficult. The process of ripping apps requires a jailbreak or Trollstore and I find it straight up impossible that anyone will ever dump this version of the game unless there's some crazy streak of luck. The new Apple TVs don't even have usb ports, or can run 32-bit apps, so just forget about those entirely. To play .tipa files on Apple TV means that you need to either sideload or install Trollstore on the thing, and to run MCSM you need an Apple TV on TVOS 10.2.2 or lower. Sideloading is something I almost never recommend. The apps only stay there for 7 days before being removed and you need to suffer through the process again. You also need a Macbook. The Reddit thread linked has mixed results and is so painful to go through, it's not nearly worth the trouble. To install Trollstore on Apple TV means that you need firmwares between 14.0 - 16.6.1, 16.7 RC (20H18) or 17.0. If you don't have these firmwares, you can still jailbreak all Apple TVs with the exception of Apple TV 4K models on TVOS 17.6.1 and above.

edit from toastito: just follow this lovely reddit post to install trollstore on your apple tv and use this lovely app to decrypt the ipa if you somehow still have a apple tv that can install trollstore and has mcsm installed

Even though no one will realistically play it, ripping the game would still help a lot in archiving this game's history. While we're here, it might be worth talking about FireTV as well. There was a FireTV release, although FireTV OS is just based off FireOS, which is based off Android, so you'd be fine installing the Android apk and hooking up a controller.

Also, for some reason the game will get stuck at the title screen if you're logged into iCloud.
